# AWS-IAM 
- IAM = Identity and Access Management, global service
- AWS root account is created by default when a user creates an AWS account
- - Should not be used or shared. Root Account is like Azure Global Admin

IAM: Permissions
1. Users or Groups can be assigned JSON documented known as IAM policy.
2. These policies define the permissions of the users.
3. In AWS, you apply the least privilege when definine a policy.
